Philosophy

Seamless user experiences are born of connected interactions between a person and an object, a user, and an interface. Naoto Fukasawa’s ‘Without a Thought’ design philosophy has it that design shouldn’t make the user work.
‍
In terms of principles of design, think ‘less is more’. Stick to the design principles of KISS: Keep It Simple, Stupid. The user interface shouldn’t contain anything that doesn’t have a purpose.
‍
To make something that people want to use, it needs to be useful. As Dieter Rams said, “good design emphasizes the usefulness of a product while disregarding anything that could possibly detract from it.”

3. User Interface

The goal of user interface design is to produce a user interface which makes it easy, efficient, and enjoyable (user-friendly) to operate a machine in the way which produces the desired result.

Outcomes

→ Creative directions
→ Mobile and desktop user interface design
→ Style guide
→ Design review
→ Developer hand-off
